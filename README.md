
# Pizzeria Information System

Автоматизированная информационная система «Пиццерия».

## Возможности
- Управление меню
- Оформление заказов
- Управление доставкой
- Авторизация пользователей
- Отчетность

## Технологии
- Backend: Python Django
- Frontend: HTML/CSS/JavaScript
- Database: PostgreSQL

## Структура проекта

```
backend/
frontend/
database/
docs/
diagrams/
```

## Запуск проекта

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```

### Frontend

Открыть файл index.html
